# Actividad 4 - Portafolio
**Nombre del proyecto:** Uso de templates

**Realizado por:** Salinas Cenobio Leonel Isaac

El propósito de esta actividad es conocer y modificar el uso de templates de diseño que ofrecen páginas como bootstrap o tailwind.

# Descripción del proyecto
Para la realización de este proyecto se esta empleado el uso de la plataofrma **Tailwind** la cual nos proporciona una amplia variedad de plantillas 
prediseñadas que podemos empler en cualquier situación en la que requeramos el diseño de una página web. En este caso la actividad va centrada a 
emplear uso de las mismas para realizar un portafolio profesional en que se deben demostrar las skills adquiridad hasta el momento, puntos de mejora y
en desarrollo. 

La plantilla utilizada la podemos obtener del siguiente enlace: **https://www.tailawesome.com/resources/atom**

# Secciones
Las secciones utilizadas de la plantilla son las siguientes:
1. **Presentación:** Pequeña presentación personal.
2. **¿Quién soy?:** Descripción de mis conocimientos.
3. **En lo que destaco:** Áreas en las que destaco.
4. **Revisa mi portafolio (Mis trabajos a través de los años):** Trabajos realizados.
5. **Clientes recientes:** Empresas en las que se trabajo con anterioridad.
6. **Contáctame:** Contacto directo.
7. **Mantente informado:** Únete al club de información.
